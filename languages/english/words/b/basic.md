# Basic

## Meaning

Basic describes someone or something that is pertaining to or constituting a base or basis.

## Examples

- A basic fact.
- The basic ingredients.
- Basic changes in public opinion occur because of changes in priorities.

## Notes

Usually used before a noun or after verbs like "be," "feel," "seem," or "look."
