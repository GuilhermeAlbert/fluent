# Character

## Meaning

Character is an imaginary person represented in a work of fiction.

## Examples

- She is the main character in the novel.
- The character asked a clear question.
- I spoke with the character yesterday.

## Notes

Usually used for a person or role. Add "a," "an," or "the" when talking about one specific person.
