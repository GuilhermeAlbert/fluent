# Container

## Meaning

Container means any object that can be used to hold things.

## Examples

- I left the container on the table.
- She picked up the container carefully.
- The container is in the kitchen.

## Notes

Usually used as a noun for a physical object. Add "a," "an," or "the" when needed.
