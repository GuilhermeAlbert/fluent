# Describe

## Meaning

To describe means to give a description of.

## Examples

- He drew an elaborate plan of attack.
- I need to describe this clearly.
- She tried to describe the idea.

## Notes

Most often used as a verb. It can appear after "to," "can," "should," or a subject.
