# Editor

## Meaning

Editor is a person responsible for the editorial aspects of publication.

## Examples

- The editor asked a clear question.
- I spoke with the editor yesterday.
- The editor joined the meeting.

## Notes

Usually used for a person or role. Add "a," "an," or "the" when talking about one specific person.
