# Function

## Meaning

Function is a mathematical relation such that each element of a given set is associated with an element of another set.

## Examples

- We talked about the function.
- The function matters here.
- I noticed the function right away.

## Notes

Commonly used as a noun. The exact article or plural form depends on the sentence.
