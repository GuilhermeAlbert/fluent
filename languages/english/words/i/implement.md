# Implement

## Meaning

To implement means to apply in a manner consistent with its purpose or design.

## Examples

- Implement a procedure.
- I need to implement soon.
- They want to implement together.

## Notes

Most often used as a verb. It can appear after "to," "can," "should," or a subject.
