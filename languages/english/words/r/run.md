# Run

## Meaning

To run means to move fast by using one's feet, with one foot off the ground at any given time.

## Examples

- The horses run across the field.
- Don't run - you'll be out of breath.
- The children ran to the store.

## Notes

Most often used as a verb. It can appear after "to," "can," "should," or a subject.
