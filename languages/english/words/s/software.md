# Software

## Meaning

Software means written programs or procedures or rules and associated documentation pertaining to the operation of a computer system and that are stored in read/write memory.

## Examples

- The market for software is expected to expand.
- I need more software before I decide.
- We talked about software after lunch.

## Notes

Usually uncountable; use "some" or "more" instead of "a" in most everyday contexts.
