# User

## Meaning

User means someone who makes use of a thing.

## Examples

- The user asked a clear question.
- I spoke with the user yesterday.
- The user joined the meeting.

## Notes

Usually used for a person or role. Add "a," "an," or "the" when talking about one specific person.
