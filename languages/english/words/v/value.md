# Value

## Meaning

Value is a numerical quantity measured or assigned or computed.

## Examples

- The value assigned was 16 milliseconds.
- We talked about the value.
- The value matters here.

## Notes

Often used for abstract ideas, feelings, qualities, or information rather than physical objects.
