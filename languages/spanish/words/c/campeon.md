# Campeón

## Meaning

Campeón means "champion, champ."

## Examples

- Hablamos del campeón.
- El campeón es importante.
- No entiendo el campeón.

## Notes

Usually used as a masculine noun. Pair it with "el" or "un" when the sentence needs an article.
