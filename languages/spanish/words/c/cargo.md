# Cargo

## Meaning

Cargo means "charge or burden."

## Examples

- Hablamos del cargo.
- El cargo es importante.
- No entiendo el cargo.

## Notes

Usually used as a masculine noun. Pair it with "el" or "un" when the sentence needs an article.
