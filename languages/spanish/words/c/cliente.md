# Cliente

## Meaning

Cliente means "client or customer."

## Examples

- Hablamos del cliente.
- El cliente es importante.
- No entiendo el cliente.

## Notes

Usually used as a masculine noun. Pair it with "el" or "un" when the sentence needs an article.
