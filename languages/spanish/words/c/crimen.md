# Crimen

## Meaning

Crimen means "violent crime."

## Examples

- Hablamos del crimen.
- El crimen es importante.
- No entiendo el crimen.

## Notes

Usually used as a masculine noun. Pair it with "el" or "un" when the sentence needs an article.
