# Cuatro

## Meaning

Cuatro means "a four-stringed guitar with the first string a fifth below or instead of a fourth above."

## Examples

- Hablamos del cuatro.
- El cuatro es importante.
- No entiendo el cuatro.

## Notes

Usually used as a masculine noun. Pair it with "el" or "un" when the sentence needs an article.
