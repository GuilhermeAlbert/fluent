# Sistema

## Meaning

Sistema means "system or scheme."

## Examples

- Hablamos del sistema.
- El sistema es importante.
- No entiendo el sistema.

## Notes

Usually used as a masculine noun. Pair it with "el" or "un" when the sentence needs an article.
